# /diagram - Agent Diagrammeur

Lance l'agent Diagrammeur pour créer des visualisations interactives.

## Instructions

1. Lis le fichier `.claude/agents/diagrammer.md`
2. Charge les fichiers research et images de `workspace/`
3. Crée les visualisations appropriées :
   - Timeline pour les sujets historiques
   - Comparaisons pour les oppositions
   - Diagrammes pour les relations
   - Cards pour les listes interactives
4. Sauvegarde dans `workspace/visuals/`

## Arguments (optionnel)

$ARGUMENTS

## Action

Crée les visualisations maintenant.
