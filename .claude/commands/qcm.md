# /qcm - Agent Générateur de QCM

Lance l'agent Générateur de QCM pour créer des questions.

## Instructions

1. Lis le fichier `.claude/agents/qcm-generator.md`
2. Charge les fichiers research et fiche de `workspace/`
3. Génère des questions avec :
   - Distracteurs plausibles
   - Feedbacks instructifs
   - Anecdotes mémorables
4. Sauvegarde dans `workspace/drafts/`

## Nombre de questions (optionnel)

$ARGUMENTS

Par défaut : 10 questions

## Action

Génère les questions maintenant.
