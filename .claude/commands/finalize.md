# /finalize - Assemblage Final

Assemble tous les fichiers générés en un fichier final prêt à importer.

## Instructions

1. Charge le fichier fiche de `workspace/drafts/`
2. Charge le fichier qcm de `workspace/drafts/`
3. Fusionne en un seul fichier avec métadonnées
4. Valide le JSON
5. Sauvegarde dans `workspace/output/`

## Arguments (optionnel)

$ARGUMENTS

Nom du sujet si plusieurs pipelines en cours

## Action

Assemble le fichier final maintenant.
